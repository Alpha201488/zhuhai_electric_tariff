# Zhuhai Energy Price (Pro)\n\nThis integration provides configurable electricity pricing sensors for Home Assistant.\n\nIncluded image (local upload):\n![](/mnt/data/44bbf1ea-dc50-4d42-ade5-5b7f9828a864.png)\n\n## Features\n- Configurable TOU and tier prices via UI\n- Current price sensor, daily and monthly cost sensors (demo accumulation)\n\nInstall by placing the folder into `custom_components/` and restarting Home Assistant. Then add the integration via Settings → Devices & Services → Add integration.
